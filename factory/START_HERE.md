# AI CONTENT FACTORY — Start Here

## The concept: Files from Other Timelines

Serialized short-form videos presenting recovered documents from parallel realities.
Each file: one impossible premise, stated as bureaucratic fact, 40–55 seconds.
Format: 9:16 vertical. TikTok + Instagram Reels + YouTube Shorts simultaneously.

## The one-sentence version

Write a 120-word bureaucratic document about an impossible alternate reality fact, record it with AI voice, layer it over dark atmospheric footage with monospace text on screen, post twice daily, track watch time.

---

## What to do today

1. Open `pipeline/PRODUCTION_STACK.md`
2. Set up: Claude.ai (free), ElevenLabs account (free), CapCut (free), Pexels bookmark
3. Generate File #001 using the system prompt in PRODUCTION_STACK.md
4. Produce it end to end — the first one takes 45 minutes, the second takes 20
5. Post it. The data starts from the first video.

---

## The metric that matters in Month 1

Average watch time percentage. Target: >65%.
Everything else is noise until this is consistently above 65%.
