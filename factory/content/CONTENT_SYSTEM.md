## 30-DAY POSTING PLAN

**Posting cadence:** 2 videos/day. 60 videos over 30 days.
**Rationale:** In the growth phase, frequency beats quality optimization. 60 data points in 30 days vs. 30 data points. Algorithm exposure doubles. Learning rate doubles.

**If 2/day feels like too much:** 1/day for 30 days = 30 data points. Still sufficient. Never go below 1/day during the first 30 days.

### Week 1 — Establish the Format (Days 1–7)

Goal: algorithm learns the content type, audience learns the format.

| Day | AM Post | PM Post | Purpose |
|---|---|---|---|
| 1 | File #001 (Biology — sleep) | File #002 (Time — 11-min gap) | Format introduction |
| 2 | File #003 (Language — 29th letter) | File #004 (Geography — ocean maps) | Category variety |
| 3 | File #005 (Impossible — room not built) | File #006 (Time — 1987) | Escalate impossibility |
| 4 | File #007 (Biology — pain delay) | File #008 (Technology — internet 1947) | Mix biological + tech |
| 5 | File #009 (Society — 3am) | File #010 (Impossible — File #088) | Meta-impossibility |
| 6 | File #011 (Biology — mirrors) | File #012 (Geography — river) | Second week preview |
| 7 | **REVIEW DAY** — analyze which files from Days 1–6 performed highest. Post the top performer again with different hook. | | Format test |

### Week 2 — Find the Resonant Category (Days 8–14)

Goal: identify which category drives the most comment engagement.

Post files that vary category deliberately. Track comment patterns by category.

| Category | Comment signal to watch |
|---|---|
| Biology | "Wait this is actually true" — fact-check instinct |
| Time | "What happened during the 11 minutes?" — mystery engagement |
| Language | "What's the 29th letter?" — answer-seeking |
| Impossible | "I've seen this" — identity projection |
| Society | "This is my city / my experience" — recognition |

By Day 14, you know which category your audience responds to most. Weeks 3–4 lean into that category while maintaining variety.

### Week 3–4 — Volume + Iteration (Days 15–30)

- Continue 2/day posting
- Introduce the first "meta" file by Day 21 (File #088, File #100)
- Test a "series within the series": 3 consecutive files about the same alternate reality, different angle
- Day 25: post File #001 again with a different hook. Compare performance to original.
- Day 28: post a "request file" — ask in comments which timeline they want documented next

## KPI TRACKING SYSTEM

Track weekly. Not daily — daily variance is noise.

### Tier 1 — Algorithm KPIs (what the platform measures)

| KPI | Definition | Target (Week 4) | Target (Month 3) |
|---|---|---|---|
| Average watch time % | Time watched / video duration | >65% | >70% |
| Completion rate | % watching full video | >45% | >55% |
| Follower conversion | New follows / 1,000 views | >15 | >20 |
| Shares per 1K views | Share actions / 1,000 views | >8 | >15 |
| Comments per 1K views | Comment actions / 1,000 views | >12 | >20 |

### Tier 2 — Content KPIs (what you measure)

| KPI | Definition | Measurement |
|---|---|---|
| Hook effectiveness | % watching past 3 seconds | TikTok analytics (3s view rate) |
| Category performance | Avg views by file category | Manual tracking in spreadsheet |
| Best hook type | Text-only vs voice-lead vs structural | A/B test weekly |
| Comment quality | % comments showing discovery behavior | Manual scan 2x/week |

### Tier 3 — Growth KPIs (what matters for revenue)

| KPI | Definition | Target |
|---|---|---|

### The One Metric That Matters Most (in Month 1)

**Average watch time percentage.**

Everything else follows from watch time. If >65% of the video is watched on average, the algorithm promotes it. If <50%, it doesn't. Every production decision in Month 1 should be evaluated against this one number.

The hook gets them in. The structure keeps them. The twist makes them rewatch.

If watch time is low: fix the structure, not the concept.
If watch time is high but views are low: fix the hook, not the content.
If watch time is high and views are high but followers are low: fix the follow trigger (call to action or series mechanic).
