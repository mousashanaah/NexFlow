# 03 — DAILY PROTOCOL

Your dream day is good and it's the right day. The job of this file is to make it the
*default* day, and to protect it from the two things that blow it up: the household
cortisol load, and the snacking willpower-drain.

## The Default Day (your template, sequenced for performance)
1. **Wake early.** Before anything: 3 min to set today's ONE practice objective in the log.
2. **Short game + putting** at the greens (skill when fresh, nerves calm).
3. **Gym — dynamic warmup / activation** (prep, not fatigue).
4. **Range** — the day's ONE objective, with tape + video. Exit on the condition, not the clock.
5. **Course — 9 to 18 holes**, carrying only the day's objective as a swing thought.
6. **Gym — real lifting session** (placed AFTER golf so it never steals from skill quality).
7. **Home: NexFlow work** + **Quran / reading** (the cold, non-golf identity anchor).
8. **Call gf, wind down, in bed early.** Sleep is a training input, not leftover time.

### Two non-negotiable guards
- **State reset before performance.** Before the course/range: 5 slow nasal breaths,
  longer exhale than inhale, drop the shoulders. You play in threat-mode; this brings the
  nervous system down so your real swing shows up. Especially on days the house was loud.
- **The house is not your nervous system's problem.** Parents fighting = noise, not signal.
  You do not arbitrate it, absorb it, or carry it to the first tee. Headphones, leave early,
  protect the morning. This is performance management, not coldness.

## The Snacking Lock (it's environment, not willpower)
The blowups happen on "certain days" — those are high-depletion days (poor sleep, stress,
long sessions). Willpower is lowest exactly when you lean on it most. So we remove the decision:
- Don't keep the trigger foods in the house/bag. You can't snack what isn't there.
- Pre-pack the day's food the night before. Decisions made fed and calm, not tired and hungry.
- Eat enough protein + carbs around training so you're not under-fueled into a crash.
- One planned treat slot if wanted — scheduled, not stolen. Kills the binge cycle.

## THE DISCIPLINE LEDGER (daily score — this is the system's heartbeat)
No narrative. Numbers and one line. Copy the template below into `logs/YYYY-MM-DD.md` each day.

```

## Set in the morning
Practice objective (ONE): ______
Tournament/round today? (y/n): ___

## Scored at night (1 = done as designed, 0 = not)
[ ] Set + hit the ONE practice objective with external feedback (tape/video)
[ ] Short game + putting done
[ ] Range done on-objective (no chasing)
[ ] Course / playing reps done
[ ] Lifting session done
[ ] NexFlow work
[ ] Quran / reading
[ ] Nutrition held (no unplanned snack blowup)
[ ] In bed on time / sleep protected
[ ] State reset used before performance

## Numbers
Discipline score:   /10
Catastrophic misses today (the north star — drive to zero): ___
Stock-shot command rate /10: ___
Sleep hours: ___    Weight (optional): ___

## One line (only one — what mattered, no essays)
______
```

## Weekly review (with the coach)
Open the 7 logs. Look at: discipline-score trend, catastrophic-miss trend, what the
objective was each day and whether it advanced the Q-School tree. Adjust the next week.
Trend beats any single day. We coach the line, not the dot.
