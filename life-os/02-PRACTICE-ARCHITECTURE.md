# 02 — PRACTICE ARCHITECTURE

You hate the range because you wander it with 40 swing thoughts and no exit condition.
Fix: every session is a controlled experiment with ONE variable, a feedback signal, and a
finish line. This is your trading system applied to your swing. You already know how to do this.

## Rule of One
Each range session has exactly ONE objective written down before the first ball.
If you change the objective mid-session, the session is logged as failed-discipline.
You don't get to chase your tail anymore. The constraint IS the skill.

## Feedback is mandatory (you cannot self-diagnose from inside the storm)
Every technical session uses at least one EXTERNAL signal so your threat-brain isn't the judge:
- **Impact tape / foot spray on the face** — shows heel/toe/low-point pattern (cheap, decisive).
- **Down-the-line video** every session — same angle, phone on a stand.
- **An alignment stick** for posture/space reference.
You are an expert diagnostician. Give yourself the same footage you'd give a client.

---

## PRIORITY ZERO PROTOCOL — Isolate the Two-Way Miss
Goal: confirm or kill the early-extension/stuck hypothesis, then fix the actual fault.
Run this before any "scoring" practice. The defect metric (catastrophic miss) outranks everything.

### Phase A — Diagnose (1-2 sessions, just gather data)
1. Hit 30 balls, normal intent, 7-iron. Face spray on the clubface every ~5 balls.
2. Film down-the-line. Log the strike pattern: where on the face, and what the miss did.
3. Tag each miss: HEEL (toward shank), or CLOSED-FACE (snap hook), or clean.
4. In video, check at impact: **has your pelvis moved toward the ball / have you stood up?**
   Compare hip line and trail-hip depth from address to impact.
Outcome: confirm (or reject) early extension as the engine. Write the verdict in the log.

### Phase B — If early extension confirmed: low-point + space drills
Pick ONE per session, log command rate /10:
- **Trail-glute-to-wall drill:** set up with trail glute touching a wall/bag; keep it back
  through the downswing. Trains pelvis to rotate, not thrust.
- **Towel-under-trail-arm / connection** for the stuck/under-plane component.
- **Line drill for low point:** spray line or board; goal is divot starting *after* the ball,
  consistently. Strike location on tape is the scoreboard.
Exit condition: 8/10 clean strikes on tape with stable posture in video → session passed.

### Phase C — Pressure-test the miss
Once clean in blocks, you must prove it survives randomness, because your miss is random:
- **Random-target, single-ball routine:** new target every ball, full pre-shot routine,
  no rakes of balls. Track catastrophic-miss count per 20. Target: zero.
- Only when zero across multiple sessions does the fault "graduate."

## Session Menu (pick the day's ONE objective from here)
- **Defect (Priority Zero):** the protocol above. Runs until catastrophic miss → ~zero.
- **Stock shot:** one shape, one target, command rate /10. Build the shot you trust at Q-School.
- **Scoring (50-125 yds):** distance ladders, proximity logged. Won-or-lost zone.
- **Short game:** up-and-down games with a score, not aimless chipping.
- **Putting:** lag (kill 3-putts) + a pressured short-putt game with a make-streak target.
- **On-course:** transfer test — play with the day's objective as the only swing thought.

## Why this beats your old practice
Old: hours, no direction, threat-brain judging, ceiling work, ego-protecting.
New: one variable, external scoreboard, floor-raising, defect-killing. The card lives here.
