# AFTERIMAGE — Consistency System
## How to produce 100+ videos without drift, contradiction, or loss of identity

---

## THE PROBLEM

Content universes collapse for one reason: **inconsistency breaks the obsessed viewer's trust.**

The casual viewer won't notice if the 23rd floor has the wrong color temperature in video 47.
The obsessed viewer will.
The obsessed viewer is the one who drives all growth, all sharing, all theories, all evangelism.
Losing the obsessed viewer means losing the engine.

The consistency system is not about perfectionism. It is about protecting the one audience that matters.

---

## LAYER 1: THE VISUAL LOCK

**The DaVinci Power Grade (the single most important consistency tool)**

Export this as a `.drx` Power Grade file after building it once. Apply to every single clip on import. Every video starts from the same visual baseline.

Settings to lock:
- Shadow lift: dark teal-gray (exact values in pipeline.md)
- Highlight tone: warm amber
- Saturation: 75-80%
- Grain overlay: 15-20% opacity, specific grain file (save and reuse — same grain every time)
- Lens vignette: subtle, consistent across all videos

**This alone handles 80% of visual consistency.**

The remaining 20% is per-Anchor color work — each Anchor has its own color note (in the Anchor Map in Notion). When shooting or grading an Anchor, pull the note and confirm you're matching.

## LAYER 2: THE LORE LOCK

**The Notion Database (minimum viable setup)**

Create 4 databases:

**DB 1: Video Log**
| Field | What to track |
|-------|--------------|
| Post # | Number in sequence |
| Title | Internal reference name (never public) |
| Type | Pure Moment / Fragment / Echo / Anchor Visit / Interruption |
| Anchors featured | Which of the 6 Anchors appears |
| Symbols present | Which symbols appear, in what state |
| Phase | 1 / 2 / 3 (canonical timeline phase of this video) |
| Lore events | What new information this contains |
| Seeds | What this plants for future videos |
| Reveals | What previous seeds this activates |
| Posted date | Date live |

**DB 2: Symbol Tracker**
| Symbol | Total appearances | States it's appeared in | Last appeared | Next planned |
|--------|------------------|------------------------|---------------|--------------|
Track every symbol here. Before any new video, check: has this symbol appeared in the last 4 posts? If yes, hold it.

**DB 3: Anchor Map**
| Anchor | Real location | Color notes | Emotional register | Last post # | Next planned |
|--------|--------------|-------------|-------------------|-------------|--------------|
One row per Anchor. Pull this before filming any Anchor.

**DB 4: Canon Events**
| Event | Internal description | Phase | Video # it seeded | Video # it revealed |
|-------|---------------------|-------|-------------------|--------------------|
The complete lore timeline — internal only. Never posted. Every event tracked here before any video hints at it.

## LAYER 3: THE PRE-POST CHECKLIST

Before publishing every video, answer these:

**Visual check:**
- [ ] DaVinci Power Grade applied?
- [ ] Anchor color match confirmed (if Anchor present)?
- [ ] Protagonist face remains unseen?
- [ ] Audio level consistent with other videos (set a reference level)?

**Lore check:**
- [ ] Does this contradict any entry in Canon Events?
- [ ] Are any symbols appearing for the first time clearly recorded in Symbol Tracker?
- [ ] If this is an Echo, have I confirmed the original video's details and matched intentionally?
- [ ] If this is an Interruption, does the Canon Doc support this as a phase event?

**Identity check:**
- [ ] Caption is a fragment, lowercase, no explanation?
- [ ] Maximum 3 hashtags?
- [ ] No creator voice visible?

If any box is unchecked: do not post until it's resolved.

---

## LAYER 4: THE 10-VIDEO AUDIT

Every 10 posts, spend 30 minutes doing a full audit:

1. Watch all 10 videos back-to-back
2. Check for: color drift, tonal drift, pacing inconsistency, unintended continuity errors
3. Check Symbol Tracker — is any symbol overused or underused?
4. Check Anchor Map — have you visited the same Anchor too many times in a row?
5. Check the lore arc — are you roughly on track with the arc map?

This takes 30 minutes every 10 posts (~monthly). It prevents the slow drift that destroys long-form universes.

---

## LAYER 5: THE RESERVE RULE

**Always have 4 finished videos in reserve.**

Never post from an empty queue. Never produce under pressure. Pressure breaks consistency.

The reserve means:
- A missed week doesn't break the cadence
- A bad video can be held without a gap in posting
- You can review the last 10 posts before releasing the next one, maintaining perspective

**How to build the reserve:** The first month of production, make 8 videos before posting any. This creates a 4-video buffer before the public timeline begins.

## LAYER 6: THE STYLE BIBLE (Quick Reference Card)

Print this. Keep it next to where you edit.

```
AFTERIMAGE STYLE BIBLE

COLORS:
  City/base: dark teal shadows, warm amber highlights
  23rd Floor: warm orange-amber (specific)
  Overpass: cold blue-gray
  Convenience Store: green-white fluorescent
  Stairwell: deep blue
  Phase 2 indicator: rain/wet surfaces
  
PROTAGONIST:
  Face: never clearly seen
  Body: from behind, small in frame
  Hands: the exception — shown clearly
  Voice: none
  
MUSIC:
  Ambient, sparse, no lyrics or non-native-language only
  Audio decay past visual cut
  
CAPTION:
  Fragment, lowercase, no hashtag spam (max 3)
  Never first person
  Never explanation
  
SYMBOLS:
  Max 1 per video
  Max 1 appearance per 4 posts
  Never explained
  
THE RULE:
  Every video ends before it's over.
```

## WHAT BREAKS UNIVERSES (Avoid All)

1. **Answering questions in comments** — one answer closes infinite theories
2. **Caption tone shift** — suddenly posting something upbeat or self-referential
3. **Rushing a lore reveal before it's seeded** — the audience needs 10 seeds before 1 reveal
4. **Changing the color grade** — even slightly — after the first 20 posts
5. **Posting more than 3x/week** — scarcity is structural, not optional
6. **A video that's "just aesthetic" with no lore function** — every video must do something, even if subtle
7. **Collab content that breaks the fictional frame** — no exceptions
8. **Explaining the universe in an interview or external post** — the account should have no external presence
